8. WHAT IS THE SIGNIFICANCE OF "%", AND "_" OPERATIONS IN THE LIKE STATEMENT?

ANS: In SQL, the "%", AND "_" symbol used with LIKE represents a wildcard for pattern matching. It allows us to find data based on specific conditions.

-If you want to find employees in the Employee table whose names start with the letter "A":

SELECT * FROM employee WHERE emp_name LIKE 'A%';

-To find employees whose names contain the letter “P” anywhere

SELECT * FROM employee WHERE emp_name LIKE '%P%';

-FIND THE EMPLOYEES WHOSE NAME IN 5 LETTERS.

SELECT *FROM EMPLOYEE WHERE NAME LIKE '_____";

11. WHAT DO YOU UNDERSTAND ABOUT DDL,DCL AND DML IN MYSQL?

ANS:  DDL - DATA DEFINATION LANGUAGE - CREATE , ALTER , DROP
